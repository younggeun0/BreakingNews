# 📰BreakingNews

* **뉴스 RSS XML를 사용해 Parsing하는 Web Toy Project**
* 간단한 **Bootstrap** 사용
* **JDOM Parser** 사용해서 **XML Parsing**
* [JTBC 뉴스 속보 RSS](http://fs.jtbc.joins.com//RSS/newsflash.xml) 이용
  * description Node가 100자 이상이면 '...'처리
